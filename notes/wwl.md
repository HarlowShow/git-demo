### Getting Status Info
`git status`

### Making Updates

##### Staging Changes
`git add [filename]`
Stage a file
`git add .`
Stage all

##### Committing Changes
`git commit -a -m "comment here"`
Commit all changes (-a), including a commit message(-m)git (you can also use `-am`)

### Getting Info/Help
`git config --list`
See your config deets

`git log [-n]`
See commit history [/latest n commits]
